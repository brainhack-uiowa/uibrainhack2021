Information

The purpose of the University of Iowa brainhack this summer is to help participants learn MATLAB fundamentals and EEG Analysis.

Iowa City Brainhack is dedicated to a harassment-free conference experience for everyone. Our anti-harassment policy can be found here: Code of Conduct.
If you have any comments/concerns please email trevor-cline@uiowa.edu, gail-harmata@uiowa.edu, marco-pipoly@uiowa.edu, ethan-rooke@uiowa.edu, kerry-tarrant@uiowa.edu, or jax-skye@uiowa.edu.

Schedule

We encourage you to have both the reading and assignment opened at the same time so you can run the code blocks as well as read the material.
June 4th
3:00 to 5:00 Install Matlab/Citrix help (as needed), intro to GUI and navigation, syntax, variables, data types
June 11th
3:00 to 5:00 Strings, vectors, arrays, matrices, colon notation, nesting
June 18th
3:00 to 5:00 Loops, decisions, operators, functions, intro to plotting
June 25th
3:00 to 5:00 Catch-up day
July 2nd
3:00 to 5:00 Theory about ephys/EEG, preprocessing, artifact identification, ICA intro
July 9th
3:00 to 5:00 ICA continued, Intro to ERP
July 16th
3:00 to 5:00 More ERP code, plotting
July 23rd
3:00 to 5:00 Catch-up day; time-frequency analysis discussion if time
